# YMM4 NVEncC 直接呼び出しプラグイン 計画

## 前提
- NVEncC を直接呼び出す
- GUI 必須項目: コーデック（h264/h265）、ビットレート
- 旧バージョン対応は行わない

## 進め方
1. プラグイン種別と出力フローの確定
   - YMM4 の動画出力プラグイン（IVideoFileWriterPlugin）前提で、出力パイプラインへの統合点を整理
2. GUI 仕様の確定
   - 必須項目（h264/h265、ビットレート）に加え、追加候補を優先度順で決定
   - 追加候補例: レート制御（CBR/VBR/QVBR/CQP）、プリセット（P1–P7）、Bフレーム数、GOP長、tune
3. NVEncC コマンド設計
   - GUI 入力 → NVEncC 引数の対応表を作成（-c, --cbr/--vbr, --preset, --bframes, --gop-len など）
4. 実装設計
   - 設定保存、NVEncC 実行、進捗/ログの取り回し、エラー表示方針を決定
5. 最小実装 → 動作確認
   - まず「h264/h265 + ビットレート」だけで出力が通ることを確認
6. 追加機能の拡張
   - 追加候補を段階的に GUI へ組み込み、バリデーションとエラーハンドリングを整備
